Mengenal apa itu front-end, back-end, dan database

Ditulis oleh M.Muizzuddin pada 22 Oktober 2020

Berikut komponen yang diperlukan dalam membuat sebuah website:

1. Front-End

Bagian front-end dari sebuah website adalah bagian yang langsung dilihat oleh user. Bagian ini dibangun menggunakan HTML,CSS, dan JavaScript.

HTML(Hyper Text Markup Language) adalah tulang punggung dari web. Semua website yang kamu kunjungi dibuat menggunakan HTML. HTML menentukan stuktur dan konten website. Versi terkhir dari HTML adalah HTML5.

CSS(Cascading Style Sheet) adalah bahasa pemrogamana yang mengontrol tampilah HTML pada halaman website. CSS menentukan warna, font, gambar, background, dan bahkan bagaimana tampilah keseluruhan website-mu. Versi terakhir dari CSS adalah CSS3, yang menambahkan fitur interaktif dan animasi dasar.

Kamu bisa menciptakan sebuah website hanya dengan menggunakan HTML dan CSS, tetapi yang benar benar akan membuat website kamu tampil menarik adalah JavaScript. Dengan Javascript, kamu bisa melakukan banyak hal, seperti interaktifitas, animasi yang lebih kompleks, dan bahkan kamu juga bisa membuat fully fiture web application

Dulu (sekitar tahun 2012), kebanyakan browser tidak dapat menginterpretasikan Javascript dengan baik sehingga menambahkan fungsi-fungsi kompleks JavaScript bukanlah ide yang bagus. Tetapi saat ini browser sudah menjadi lebih canggih dan bisa menjalankan JavaScript dengan baik. Ini menjadikan JavaScript tidak hanya digunakan untuk bagian back-end lagi. JavaScript sendiri juuga telah mengalami banyak kemajuan dengan bertambahnya framework-framework seperti Angulars, JQuery, dan Node.js. Singakatnya, apa yang didefinisikan dengan front-end telah banyak berubah dalam waktu yang singkat.

front-end developer menggunakan HTML, CSS, dan JavaScript untuk coding website mereka menciptakan design wesite dan kemudian membuat website tersebut dapat berfungsi

sebagian website hanya dapat menggunakan HTML, CSS, dan JavaScript. Tetapi sebagian lainnya juga ada menggunakan back-end progamming leanguage.


2.Back-End

back-end adalah bagian belakang layar dari sebuah website. Bahasa pemograman untuk back-end developer diantaranya adalah PHP, Ruby, Python, dan banyak lainnya.

satu hal yang akan kamu temui tidak akan banyak perusahaan yang mem-posting lowongan kerja dengan kalimat "back-end developer". Yang akan lebih sering kamu temui adalah lowongan kerja untuk "Ruby developer", "PHP developer", dll.

Ada banyak hal yang tidak bisa dilakukan oleh JavaScript,namun dapat dilakukan oleh bahasa pemograman back-end. Kebanyakan sistem menejemen konten dibangun melalui back-end progamming, contohnya pada aplikasi web yang kompleks. Belajar coding akan mengajarkanmu menemukan solusi terbaik untuk berbagai masalah, dan terkadang back-end progamming adalah bagian dari solusi tersebut.


3. Database

Pengertian Database atau basis data adalah kumpulan berbagai data dan informasi yang tersimpan dan tersusun didalam komputer secara sistimatik yang dapat diperikasa, diolah, atau dimanipulasi dengan menggunakan progam komputer untuk mendapatkan infotmasi basis data tersebut.

Perangkat lunak yang digunakan untuk mengelola dan memanggil data base disebut dengan sistem database management system.

Istilah data base sendiri mengacu pada koleksi data-data yang saling terkait satu sama lain dimana tujuan data base adalah dapat digunakan untuk mengelola data dengan lebih efektif dan efisien.

Contoh aplikasi data base yang banyak digunakan adalah MySQL, Oracle SQL, Microsoft SQL Server, PostgresSQL.

Dalam website yang telah kita daftarkan, data base berfungsi untuk menyimpan data website, seperti post,page, ataupun keterangan lain yang bersifat dinamis.
